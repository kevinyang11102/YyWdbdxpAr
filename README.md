# 社区爱心养老管理系统 python493

## 项目概述

社区爱心养老管理系统是一个基于Python与Vue.js开发的系统，旨在通过有效的组织与协调，提升社区养老服务的质量和效率，为老年人提供便捷、温馨的服务。

## 技术栈

- **后端语言**: Python 3.6.8+
- **后端框架**: Django
- **前端**: Vue.js + HTML
- **数据库**: MySQL
- **编译器**: PyCharm

## 功能模块

- 用户管理
- 身体健康监测
- 公共书籍管理
  - 图书分类
  - 借阅信息
  - 归还信息
  - 还书入库
- 活动信息发布
  - 活动分类
  - 活动报名
- 系统管理
- 个人信息管理

## 系统角色

- 系统管理员
- 普通用户（老年人）
- 服务人员

## 运行环境

- Python 3.6.8 或更高版本
- Django 框架
- Vue.js
- MySQL 数据库
- PyCharm 或其他兼容的IDE

## 部署步骤

1. 安装Python环境和MySQL数据库。
2. 克隆或下载项目源代码。
3. 配置数据库连接。
4. 安装依赖项（使用pip或npm）。
5. 运行项目。

